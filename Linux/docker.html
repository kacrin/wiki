docker 容器

新建并启动容器使用 docker run

使用容器输出hello world 然后终止容器

# docker run ubuntu:14.04 /bin/echo 'hello world'

启用一个bash终端，交互运行

# docker run -t -i ubuntu:14.04 /bin/bash

使用docker start ,直接使已终止容器运行
